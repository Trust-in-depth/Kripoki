# Kripoki Frontend

Bu klasör, Kripoki projesinin React + Vite tabanlı frontend uygulamasını içerir.

Tam kurulum, Supabase ayarları ve çalıştırma adımları için depo kökündeki ana README dosyasına bakın:

- [Ana README](../README.md)

## Hızlı Başlangıç

```powershell
cd audit-shield
npm install
Copy-Item .env.example .env
npm run dev
```

`.env` dosyası içine kendi Supabase değerlerinizi eklemeyi unutmayın.
